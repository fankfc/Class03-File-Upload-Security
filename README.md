# Class03 Flask 文件上传功能实现与安全加固

> **项目名称：** Class03-File-Upload-Security  
> **日期：** 2026-07-21  
> **仓库地址：** `github.com/fankfc/Class03-File-Upload-Security`

---

## 项目概述

基于 Flask 的用户管理平台，在登录、注册、搜索功能基础上，新增**头像上传功能**并实施完整的**4 层安全防护**，防御 OWASP Top 10 中常见的文件上传漏洞。

## 安全防护架构

```
用户上传文件
    │
    ▼
┌──────────────────────────────────┐
│ Layer 1: 扩展名白名单             │
│ 仅允许: .jpg/.jpeg/.png/.gif/webp │ ← 拦截 .py/.html/.exe 等
└──────────────┬───────────────────┘
               ▼
┌──────────────────────────────────┐
│ Layer 2: PIL 图片内容校验          │
│ Image.open() + verify()          │ ← 拦截伪装图片（文本改后缀）
└──────────────┬───────────────────┘
               ▼
┌──────────────────────────────────┐
│ Layer 3: UUID 重命名              │
│ uuid.uuid4() + 原扩展名           │ ← 防止路径遍历 + 文件覆盖
└──────────────┬───────────────────┘
               ▼
┌──────────────────────────────────┐
│ Layer 4: 上传速率限制              │
│ 最多 3 次/分钟（独立计数器）       │ ← 防止批量上传 DoS
└──────────────┬───────────────────┘
               ▼
          ✅ 安全存储
     static/uploads/uuid.ext
```

## 功能清单

| 功能 | 路由 | 说明 |
|------|------|------|
| 首页 | `GET /` | 展示用户信息 / 搜索用户 |
| 登录 | `GET+POST /login` | CSRF + 速率限制 + 账号锁定 |
| 注册 | `GET+POST /register` | 参数化查询防 SQL 注入 |
| 搜索 | `GET /search` | 参数化 LIKE 查询 |
| 上传头像 | `GET+POST /upload` | 4 层安全防护（需登录） |
| 登出 | `GET /logout` | 清除 session |

## 安全测试结果

| 测试场景 | 攻击方式 | 结果 |
|---------|---------|------|
| 上传 .py 脚本 | 上传 Web Shell | ✅ 扩展名白名单拦截 |
| 上传 .html 文件 | 存储型 XSS | ✅ 扩展名白名单拦截 |
| 文本文件改 .png 后缀 | 伪装图片 | ✅ PIL 内容校验拦截 |
| 路径遍历 `../../app.py` | 覆盖主程序 | ✅ UUID 重命名防御 |
| 连续批量上传 | 磁盘 DoS | ✅ 速率限制拦截（3次/分钟） |
| 正常上传 PNG 图片 | 合法使用 | ✅ 成功，返回 UUID URL |
